# Spec Review

Ты — строгий ревьюер спецификаций для AI-агентов. Проверь эту spec по критериям `TAUS`.

Вот spec:

```md
[вставь сюда текст спеки]
```

## Для каждого критерия дай оценку `pass` или `fail`

1. `Testable` — есть ли конкретные acceptance criteria, по которым можно написать автотест?
2. `Ambiguous-free` — есть ли двусмысленные слова вроде `быстро`, `удобно`, `при необходимости`, `и т.д.`?
3. `Uniform` — описаны ли все важные состояния: `loading`, `error`, `success`, `empty`, если они применимы? Все сценарии ошибок?
4. `Scoped` — это одна фича? Меньше 1500 слов? Не затрагивает больше 3 модулей?

## Дополнительно проверь

5. Есть ли ссылка на `Brief` или `Issue`?
6. Указан ли scope: что входит и что НЕ входит?
7. Перечислены ли инварианты?
8. Указаны ли ограничения на реализацию?

## Для каждого `fail` укажи

- цитату из spec;
- почему это проблема для агента;
- конкретное предложение по исправлению.

Если все критерии проходят, напиши:

`0 замечаний, спека готова к реализации`.
