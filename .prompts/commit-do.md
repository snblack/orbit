# Commit Do

Сделай коммит со всеми изменениями и напиши понятный короткий `commit message` на русском языке, не упоминая, что помощником была LLM.

Также, если в папке `memory-bank/features/` есть изменения, связанные с этой реализацией, добавь их в коммит.

## Дополнительные требования

- Перед коммитом проверь `git status`, `git diff` и последние commit messages, чтобы сохранить стиль репозитория.
- Не включай в коммит нерелевантные изменения, если они не связаны с текущей реализацией.
- Если есть сомнения, какие файлы относятся к реализации, коротко перечисли их и объясни выбор.
- Commit message должен быть коротким, понятным и отражать суть изменения.
